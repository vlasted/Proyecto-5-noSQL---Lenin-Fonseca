# Proyecto 5 - noSQL

API REST de películas desarrollada con Express y MongoDB.

## Tecnologías
- Node.js
- Express
- MongoDB
- Mongoose

## Estructura
- `index.js`: arranque del servidor
- `utils/db.js`: conexión a MongoDB
- `models/`: modelos de datos
- `routes/`: rutas de la API
- `seeds/`: semillas de datos
- `docs/postman/`: capturas de pruebas
